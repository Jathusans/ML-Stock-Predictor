# ML-Stock-Predictor
Machine learning-based stock predictor using technical indicators, sentiment analysis, and Random Forest to forecast short-term price direction
# Machine Learning Stock Predictor

## Overview
This project applies machine learning techniques to predict short-term stock price direction using historical data and engineered features.

The model integrates technical indicators, lagged features, and sentiment analysis to improve predictive performance.

## Features
- Feature engineering (returns, moving averages, RSI, volatility)
- Lag-based time series features
- Sentiment analysis using NLP (TextBlob)
- Random Forest classification model
- Model evaluation using accuracy and feature importance

## Model
- Random Forest Classifier
- Time-series split for training/testing
- Prediction of next-day price direction

## Outputs
- Model accuracy
- Feature importance ranking
- Probability-based predictions
- 7-day forecast simulation

## Technologies
- Python
- pandas, NumPy
- scikit-learn
- TextBlob
- matplotlib
